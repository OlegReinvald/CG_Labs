# Лабораторная 4 — отсечение (вариант 11)

Вариант 11:  
- Часть 1: отсечение отрезков прямоугольным окном алгоритмом Лианга–Барски (parametric).  
- Часть 2: отсечение выпуклого многоугольника (Sutherland–Hodgman) прямоугольным окном.

## Что делает приложение
- Рисует систему координат с масштабируемой сеткой.
- Показывает прямоугольник отсечения.
- Загружает или задаёт вручную отрезки; выделяет видимые части после отсечения.
- Отсекает выпуклый многоугольник окном; выводит результат.
- Показывает время работы алгоритмов.

## Формат входного файла (для загрузки отрезков)
```
n
X1 Y1 X2 Y2
...
Xn1 Yn1 Xn2 Yn2
Xmin Ymin Xmax Ymax
```

## Как пользоваться
1. Задайте окно отсечения: Xmin, Ymin, Xmax, Ymax.  
2. Введите отрезки (по умолчанию подставлены демо) или загрузите файл.  
3. Нажмите «Отсечь отрезки» — оригинальные отрезки серые, видимые части ярко-оранжевые. Внизу — время и число видимых.  
4. Для многоугольника: отредактируйте точки (формат `x1 y1, x2 y2, ...`), нажмите «Отсечь многоугольник».  
5. Кнопка «Очистить» сбрасывает холст.

## Запуск
```bash
python app.py
```

Нужен Python 3 с Tkinter (обычно `python3-tk`).

## Структура
- `app.py` — реализация алгоритмов, GUI.
