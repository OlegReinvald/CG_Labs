# Лабораторная 5 — 3D буква «Р» (Reinvald)

Приложение рисует объёмную букву «R/Р» в 3D, позволяет её переносить, вращать и масштабировать.

## Управление
- Слайдеры/спинбоксы: перенос по X/Y/Z, вращения по X/Y/Z (градусы), масштаб.
- Кнопка «Сбросить» — вернуть модель в исходное положение.
- Ползунок «Поворот камеры» — азимут обзора.

## Как устроено
- Контур буквы задан в 2D, экструдирован по оси Z в призматику.
- Отрисовка: ортографическая проекция, сортировка граней по средней глубине (painter) для простого скрытия невидимых.
- Грани закрашены с лёгким затенением по нормали.

## Запуск
```bash
python app.py
```

Python 3 + Tkinter.

- `app.py` — модель буквы, математика трансформаций, GUI/рендер.
