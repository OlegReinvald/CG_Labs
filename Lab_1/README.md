# Лабораторная 1 — цветовые модели (CMYK / RGB / HLS)

Простое настольное приложение на Tkinter для варианта CMYK-RGB-HLS. Можно выбрать цвет и смотреть, как он выглядит сразу в трёх моделях; любые изменения пересчитываются автоматом.

## Что умеет
- Поля ввода и ползунки для каждого канала в трёх моделях.
- Выбор цвета через системный color picker.
- Мгновенный пересчёт во всех моделях при изменении любого канала.
- Превью цвета, HEX-код и текущие значения RGB.

## Как устроено
- Конверсии: `rgb_to_hls` / `hls_to_rgb` и `rgb_to_cmyk` / `cmyk_to_rgb` с ограничением значений через `clamp`.
- Интерфейс: три карточки (RGB, HLS, CMYK) с полями и ползунками; событие на ввод и движение ползунка гоняет пересчёт.
- Логика синхронизации: флаг `updating` защищает от циклов при обновлении контролов.

## Запуск
Нужен Python 3 и Tkinter (обычно ставится вместе с `python3-tk`).

```bash
python app.py
```
