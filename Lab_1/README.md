# Лабораторная работа 1: Цветовые модели (CMYK · RGB · HLS)

## Задача
Изучить преобразования между цветовыми моделями CMYK, RGB и HLS (вариант для нечётных — CMYK-RGB-HLS). Создать приложение, в котором пользователь может выбирать цвет и видеть его представление во всех трёх моделях с автоматическим пересчётом.

## Требования из задания
- Корректный перевод между моделями — 40 баллов.
- Дружелюбный и удобный интерфейс — 20 баллов.
- Задание цвета в каждой из трёх моделей тремя способами — 20 баллов.
- Автоматический пересчёт во всех моделях при изменении любой координаты — 20 баллов.

## Реализация
- **Интерфейс**: Tkinter-приложение с тремя карточками для моделей RGB, HLS и CMYK. У каждой модели есть поля ввода и ползунки.
- **Способы задания цвета**:
  - Точное значение: поля ввода для каждого канала (Enter или выход из поля).
  - Плавное изменение: ползунки для каналов.
  - Выбор из палитры: системный color picker.
- **Автоматический пересчёт**: при изменении любой модели выполняются конверсии в две другие, UI обновляется синхронно.
- **Отображение**: превью цвета, HEX код, текущие значения RGB.

## Конверсии
- `rgb_to_hls` / `hls_to_rgb` — по классической формуле HLS.
- `rgb_to_cmyk` / `cmyk_to_rgb` — с нормализацией и учётом ключевой компоненты K.
- Вспомогательная функция `clamp` ограничивает значения допустимыми диапазонами.

## Запуск
Требуется Python 3 с модулем Tkinter (обычно ставится вместе с python3-tk).

```bash
python app.py
```
